# MLN Project - The Evolution of Production

A comprehensive, unified minimalist website portfolio UI design with integrated, synchronous anti-gravity effects across all five distinct sections, viewed as a single, continuous user journey.

## Folder Structure

Following a clean, modular structure:
- `public/`: Static files (favicon, etc.)
- `src/assets/`: Fonts and images
- `src/css/`: BEM-style modular CSS
- `src/js/`: Logic, components, and animations.
